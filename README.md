# kadai7
キャッシュについて理解する

## 実行方法
1. ローカルにクローン

2. クローンフォルダに移動

3. npm install

4. 下記コマンド実行
```
$ node ./bin/www
```

5. ブラウザで`http://localhost:3000`にアクセスする  
  
下記４パターンを確認可能です。  
**キャッシュ有り**  
[max-age600]：  
　10分間キャッシュが有効になるように設定。  
　期間中はキャッシュが利用されるため"Random Number"項目への変更は反映されません。  
  
**キャッシュ利用確認有り**  
[no-cache][max-age0]  
　必ずサーバーにキャッシュの有効性確認を行います。  
　画像とCSSは変更がないため304が返却されてキャッシュが利用されます。    
　htmlファイルはリクエストのたびに"Random Number"項目が更新されているため、  
　キャッシュは有効ではないと判定され、キャッシュは使用されません。  

**キャッシュ無し**  
[no-store]  
　キャッシュされません。
